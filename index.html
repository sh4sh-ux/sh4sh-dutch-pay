사용자설명서

01
모임 입력

모임 타이틀과 일자를 간단히 적어두세요.

02
인원 추가

+ 버튼으로 친구 이름을 한 명씩 추가합니다.

03
지출 기록

장소, 금액, 결제자, 참여 인원을 선택하면 자동 계산됩니다.

Install Guide

홈 화면에 추가

설치하면 Safari 주소창 없이 앱처럼 더 편하게 사용할 수 있습니다.

AOS
Android / Chrome 바로 설치

상단의 “앱 설치” 버튼을 누르면 설치창이 열립니다. 설치창이 안 뜨면 Chrome에서 새로고침 후 다시 눌러주세요.

ALT
버튼이 안 될 때

Chrome 오른쪽 위 ⋮ 메뉴 → “앱 설치” 또는 “홈 화면에 추가”를 누르세요.

iOS
iPhone / Safari

하단 공유 버튼 → “홈 화면에 추가” → 추가를 누르세요.
